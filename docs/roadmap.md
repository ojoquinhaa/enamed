# Roadmap recomendado

## Curto prazo
- Exportacao de recortes filtrados (CSV/Excel).
- Adicao de indicadores por categoria administrativa.
- Melhorias na densidade da tabela (colunas configuraveis).

## Medio prazo
- Historico multi-ano para comparacoes.
- Indicadores de tendencia por UF e IES.
- Logs de atualizacao da base e trilha de auditoria.

## Longo prazo
- API publica interna para integracoes.
- Dashboards comparativos entre areas de avaliacao.
- Camada de qualidade de dados e alertas automaticos.
