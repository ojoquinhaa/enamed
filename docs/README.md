# Documentacao ENAMED 2025

Este diretorio concentra a visao estrategica, os detalhes da base de dados e as decisoes de UI/UX do dashboard do ENAMED Medicina 2025.

## Indice

- `docs/visao-geral.md` - proposito, objetivos e metricas-chave.
- `docs/dados.md` - estrutura da planilha, definicao dos campos e premissas.
- `docs/arquitetura-next.md` - organizacao tecnica em Next.js (client + server).
- `docs/ui-ux.md` - diretrizes visuais, padroes de interface e comportamento.
- `docs/roadmap.md` - proximos passos recomendados e expansao do produto.
