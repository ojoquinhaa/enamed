# UI/UX e linguagem visual

## Direcao criativa
- Estilo: Data-Driven Minimal / Healthcare Institutional.
- Interface limpa, institucional e profissional.
- Sem excesso de ornamentos; foco em clareza e confianca.

## Paleta oficial
- Primaria: Verde medico `#0F766E`
- Secundaria: Verde claro `#5EEAD4`
- Destaque positivo: Verde sucesso `#16A34A`
- Texto: Cinza carvao `#0F172A`
- Fundo: Cinza muito claro `#F8FAFC`
- Bordas: Cinza suave `#E2E8F0`

## Tipografia
- Sans: IBM Plex Sans para textos e dados.
- Serif: IBM Plex Serif para titulos e destaques.

## Layout
- Sidebar fixa no desktop com navegacao e metadados.
- Navbar responsiva no mobile com acesso rapido.
- Cards com borda suave e fundo claro para hierarquia visual.

## Componentes-chave
- Barra de filtros com selects e busca rapida.
- Cards de KPI com informacao sintetica.
- Graficos de distribuicao com barras horizontais.
- Tabela completa com ordenacao e paginacao.

## Micro-interacoes
- Animacao de entrada (fade-up) em cards.
- Sombras suaves para reforcar elevacao.
- Destaques de foco para inputs e selects.

## Responsividade
- Grid adaptativo para 1 a 4 colunas.
- Tabela com scroll horizontal para telas menores.
- Sidebar substituida por nav compacto no mobile.
